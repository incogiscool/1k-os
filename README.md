# My own OS

Learning operating systems from https://operating-system-in-1000-lines.vercel.app/en/01-setting-up-development-environment

## Features

- Multitasking, switch between processes to allow multiple apps to share the CPU
- Exception Handling, Handle events requiring OS intervention, like illegal instructions
- Paging - Isolated memory address space for each app
- Sys Calls - Allow apps to call kernel features
- Device Drivers - Abstract hardware functionalities like disk read/write
- File System - Manage files on disk
- CLI - User interface for us mere humans
